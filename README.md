## SVG-sweeper
A minesweeper-clone implemented in a single SVG file. No JavaScript!

To try it just click the image below ⬇️
<img src="https://github.com/Kesefon/svg-sweeper/raw/refs/heads/master/mine.svg" width="2000" height="600" />

## Generate new games

To generate a random new game run `./gen.nim <x> <y> > mine.svg` with x & y defining the size.  
→ This requieres [Nim](https://nim-lang.org/) to be installed.  
Note: If you use Windows this won't work ootb. Good Luck.
